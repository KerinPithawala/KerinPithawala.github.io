REVIEW PAPER ON Comparative study on hierarchical and density based methods of
clustering using data analysis

Description

Review paper presented in International Conference on IoT based Control Networks and Intelligent Systems (ICICNIS 2020)

Data Mining is extraction of useful patterns using different techniques. Clustering is one such technique. In clustering, we extract all clusters from the data to find some useful information. Hierarchical clustering and density-based clustering are two such methodologies to find clusters. Hierarchical gives its results in a dendrogram using hierarchical clustering algorithm. DBSCAN is a density-based algorithm which helps find arbitrary shaped clusters. A more efficient implementation of these algorithms has been detailed in this paper.

Description

This project was made as part of Tech+Research track of Technica 2020.

My team provided feminist context for the data sets Health Behavior in School Aged Children (HBSC, hbsc.org) and Youth Behavior & Health Survey. We used data visualization and other visual comparisons to show particular areas of interest in the data set and reenvision what is missing or biased in the data to better address issues of power and inequity.
